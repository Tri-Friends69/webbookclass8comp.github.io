  • FTP(File Transfer Protocol)

    • This protocol is generally used for transfer of files from one computer to another.

    It is commonly used for Uploading and Downloading files from a web server.

Downloading

Transfer of files server to the client computer.

Uploading

Transfer of files from client computer to the web hosting server.

  • HTTP(Hyper Text Transfer Protocol)

    • This Protocols Defines the rules to be followed while transfer of information(web document) containing text,graphics,hypertext,sound,video on WWW(World Wide Web).

    It is most widely used protocol by WWW used for transfering and viewing files(also called Web pages).

    HTTP protocol was first defined by Tim Berners Lee.
