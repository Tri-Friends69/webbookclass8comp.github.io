IT Quiz

Question 1: What does HTML stand for?

a) Hyper Text Markup Language
b) High Tech Machine Learning
c) Hyper Transfer Markup Language
d) Home Text Management Language

Correct Answer: a) Hyper Text Markup Language

Question 2: Which HTML tag is used to define the body of an HTML document?

a) <header>
b) <body>
c) <main>
d) <content>

Correct Answer: b) <body>

Question 3: Which tag is used to create a hyperlink in HTML?

a) <link>
b) <url>
c) <a>
d) <href>

Correct Answer: c) <a>

Question 4: Which tag is used to add a line break in HTML?

a) <line>
b) <break>
c) <lb>
d) <br>

Correct Answer: d) <br>

Question 5: Which HTML tag is used to create an ordered list?

a) <ul>
b) <li>
c) <ol>
d) <dl>

Correct Answer: c) <ol>

Question 6: What does the <img> tag do in HTML?

a) Define an image
b) Create a link
c) Display a video
d) Insert a table

Correct Answer: a) Define an image

Question 7: Which tag is used for creating an HTML table?

a) <table>
b) <tab>
c) <tbl>
d) <tabular>

Correct Answer: a) <table>

Question 8: What is the correct way to comment out multiple lines in HTML?

a) <!-- This is a comment -->
b) / This is a comment /
c) /* This is a comment */
d) -- This is a comment --

Correct Answer: a) <!-- This is a comment -->

Question 9:Choose the correct HTML element for the largest heading:

a) <heading>
b)<head>
c) <h6>
d)<h1>

Correct Answer: d) <h1>

Question 10:Choose the correct HTML element to define important text

a)<strong>
b) <i>
c) <important>
