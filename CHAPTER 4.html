Img Tag

  • In HTML, images are inserted using < img > tag.
  • The < img > tag is an empty tag.
  • There are three types of image files which can be used in HTML, indicated by different file name extensions: .jpg, .gif and .png.

The img element has a number of attributes:

Required Attributes Img Tag

Attribute Description Possible values
src The location/path of the image files to be inserted on the webpage URL
alt Text Specifies alternate text
