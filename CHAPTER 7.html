Introductions to QBasic:-
  • What is the full form of QBasic?
    • Ans:-Quick Beginners All-Purpose Symbolic Instructions Code.
  • Who Developed BASIC?
    • Ans:-John G.Kemeny and Thomas E.Kurtz.
  • In which year it was developed?
    • Ans:-1964

  • QBasic and its Advantages

QBasic is an Integrated Development Environment(IDE), developed by Microsoft to type, edit, debug and execute BASIC language programs.

Some of the advantages of the languages are:-

  • Simple to use and learn computer languages.
  • QBasic is free and works on most computers.
  • The rules of the languages are very Simple.
  • Popular editions are GW BASIC, BASICA, QBasic and QB64.

What do ou mean by the term "Statement"?

Ans:-Commands or instructions in QBasic are called Statement.

What do ou mean by the term "Syntax"?

Ans:-Statements should be written accordance with the specific setnof the rules of the language, which are called "Syntax".

The elements required to construct a QBasic program are:

  1. Set of Characters
  2. Keywords
  3. Constants
  4. Variables
  5. Operators and expressions.

  • Character set

A character set is a defined as set of symbols used by a proramming language.

QBasic uses the given symbols as its character set.

Alphabets A,B,C,D......a,b,c,d......
Numbers 1,2,3,4,5,6,7,8,9,0
Special Characters !,@,#,$.........

  • Keyword

Keywords are those words which have special meaning in QBasic, these keywords are also known as "Reserved Words". For Example:-CLS, REM, INPUT, LET, PRINT, FOR, NEXT, IF, THEN, ELSE, ELSEIF, END, MIDS, ASC, SQR, LEN, LEFTS, TIMES and INT. These words cannot be used by programmer as constants or variables.

  • Constants

Constants are the ata or the values in a program that cannot be changed durng the program execution.

These Constants can grouped into 2 categories:-

  1. String Constant

    2. It can be a letter, word(s), numbers, combination of letters with numbers or special characters enclosed in double quotes.

    Mathe matical Expressions cannot be performed on string constants.

    "B", "APPLE", "SYMBOL NO:10205", "!!! Welcome to QBasic World !!!",etc. are some examples of string constants.

  2. Numeric Constant

    3. Numeric Constant refers t a number. A number with or without decimal point is a numeric constant. It can be positive or negative.

    Separators are not allowed to be used in numeric constant.

    Numeric data should not be enclosed in double quotes.

    Mathematical operations and logical operations can be performed on the numeric constants.

    101, 105.23, 420, 12345, etc are some examples of the numeric constants.

  • Variable

A variable is a place in the computer memory which has a name and stores data temprarily.

There are mainly 2 types of vaibles to store 2 types of constnts. They are:

  1. String Variable eg. Name$, address$ etc.
  2. Numeric Variable eg. Age%, Marks, Number1, Number2 etc.
A string variable stores string data(String Constant).
A numeric variable stores numeric data (Numeric Constant).

Rules for Naming a Variable:-

  • A variable name must begin with a letter.
  • Variable names can have alphabets and numbers.
  • No special character is allowed in the name except the Underscore.
  • in case of string variables, variable name is followed by '$' sign. Eg. Adderess$, Name$ etc.
  • Space is not allowed in variable name.
  • Reserved words cannot be used as variable names.

  1. Operator

    2. Operators are symbols that indicate the type of operation QBasic has to perform on the data or on the data or on the values of variables.
     There are four types of operators in QBasic. They are: a)Arithmetic Operations
    b)Logical Operations
    c)Relational Operations
    d)String Operations

  2. Input Statement

    3. The INPUT statement is used to accept a data value from the user and store it in a variable.
    this statement asks use to enter data by displaying a question mark(?) during the execution of the program. The program waits for user to enter the desired value through keyboard and subsquent pressing of the ENTER key once data is entered.
While using INPUT statement, the computer performs following tasks:
#Stops further execution of the program.
#It prints a question mark on the screen.
#It waits for the user to type in value and press ENTER.
#It stores the typed value into corresponding Variable mentioned in the INPUT statement.
