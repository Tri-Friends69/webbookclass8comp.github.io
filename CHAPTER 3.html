LISTS AND TABLES IN HTML

Lists

Lists are of two types-

  • ul-unordered list
  • ol-ordered list

Unordered list

  • An unordered list is a collection of related items that have no special order
  • This list is created by using <UL> tag.
  • Each item in the list is marked with a bullet.
  • Before every individual item we have to use <LI> tag.
  • Different designs of Bullets can be used in a list.
  • Default bullet style is disc.
  • It is a container tag. i.e. at the end of the list we have to type lt;/UL>.

Ordered lists

  • An ordered list is a collection of related items that are numbered.
  • This list is created by using <OL> tag.
  • Each item in the list is marked with a number/alphabet.
  • Before every individual item we have to use <LI> tag.
  • Different styles of numbering can be used in a list.
  • It is a container tag. i.e. at the end of the list we have to type </OL>.

Nesting

Using of list inside list is known as nesting.
E.g-

  • This is list 1.
    1. This is list 2.
      • It showed the use of list under list...

Table Tag

  • The < TABLE > tag allows us to arrange data into rows and columns.
  • It is a container tag.

Table Row

  • Each row in a table is defined with < TR >.
  • It is a container tag.
  • Hence, it must end with< /TR >.

Table Heading

  • Table heading can be defined using < TH > tag.
  • It displays the headings in bold letters.

Attributes Table Tag

Attribute Value Description
Bgcolor color name Specifies the background color for a table.
Border 0 or 1 Specifies whether a border around the table is required or not.
Cellpadding Pixels Specifies the space between the cell wall and the cell content.
Cellspacing Pixels Specifies the space between cells
Height, Width Pixels % Specifies the height and width of a table.
