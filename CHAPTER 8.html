Looping in QBasic & Project Handling

LOOPING

LOOPING is represented execution of certain steps (statements forming Body of Loop) for a fixed number of times". This controlled repetition can be achieved by following steps:

  1. Initailization of Control Variable
  2. Statements to be repeated forming BODY of Loop
  3. Incrementing the value of Control Variable (COUNT = COUNT + 1)
  4. Checking Control variable against FINAl (COUNT < FINAL)
    • If Condition is TRUE control goes back to the BODY of LOOP
    • Tf Condition is FALSE Control moves to next step (Statement 5)
  5. Display "LOOP OVER"
  6. END

FOR-NEXT STATEMENT

The process of looping can be simplified with the help of OFR-NEXT statement.This Single statement has all the three steps required for making a loop,i.e

  1. Initailised of control Variable
  2. Condition Checking with Final value
  3. Incrementing/decrementing Control Variable
FOR-NEXt is best suited when some statements are to be repeated fixed number of times.

How FOR-NEXT works?

  • When control comes to FOR statement first time,the Initail Value is assigned to control variable.
  • Statements given betweenFor and NEXT are executed.
  • NEXT statement increments the value of CONTrol Variable by 1 and shifts the control to FOR statement (Beginning of LOOP)
  • value of Control VAriavble is Checked against Final Value, if it is still less than or equal to Final value the statements in Body of Loop are represented.
  • If value of Control Variable exceeds thr Final Value, the control shilfts out of the loop and statement given after NEXT is executed

CREATING THE FUTURE PROJECTS

What is Project?
Any work which has a specified aim and has to be completed in defined period of time can be carried over like a project.
A project is any systematic activity which has to be completed in a given time frame.
    Every project has certain characteristics:-
  • Every projects has a defined objectives/ goal/ aim.
  • It consists of sequence of tasks and subtasks.
  • Every task has a specific purpose and is clearly defined.
  • The project has a defined timeline according to which its tasks should be completed.
  • Every project has its time limit.

PROJECT MANAGEMENT CYCLE

A project will be successfully completed if we follow the steps of project management cycle.
A Project Management Cycle consists of following steps:
  • Identification and Planning: The main aim of the project is identified and planning is done to achieve the aim in a defined time period.
  • Coordination and Organization: Proper coordination of all the activities is done to complete each task in time . Finally, the tasks are compiled to get the final project
  • Implementation: the project is executed after compiling all the activities.
  • Monitoring the tasks are supervised thoroughly so that the activities are running properly.
  • Evalution and Testing All the tasks are evaluted and tested properly before finally executing the project.

What is fragmentation?
Whenever we store a file on a computer it takes some space in the memeory(hard disk). We mat store more than one files in a computer. These files are stored one after another in the computer's memory. After some time if we don't require a file, we may delete it . This leaves a blank chunk of memory. These chunks are called Fragments.

Defragmentation is the process of reorganizing the computer memory to arrange all the files in countinuity and combine all the blank chunks together to form a large blank space in memory.
